Privacy Policy

Transcendence Pong stores account, profile, session, match, friendship and chat data needed to run multiplayer games.

Passwords are hashed before storage. Session tokens are stored as hashes. Two-factor secrets are encrypted and recovery codes are stored as hashes.

Match history, leaderboard data and lobby messages are visible to authenticated users. Direct messages are visible only to the participants, except where required for moderation by project administrators.

Users can update profile data, export relevant account data through project-maintained endpoints, and request deletion through the administrators of this deployment.

No production API keys or secrets are committed to the repository. Runtime secrets must be stored in the local environment file.
